# Simple-Hotel-Management-Project
This is a simple hotel management system written in python.

Password is: INT213 

You can change this password to your wish in main.py

Requirements:

1. MySQL
2. Python3
3. tkinter module in python3
4. mysql-connector


Note: You have to change your mysql username and password in the following files before you run the program:
1. create.py
2. input_data_gui.py
3. display_data_gui.py


If you run this program for the first time you have to run "create.py" file first:

python3 create.py

This will create a database and a table in your mysql.

After that you can run:

python3 main.py

Note: You have to run create.py only once when you will run program for fist time.
